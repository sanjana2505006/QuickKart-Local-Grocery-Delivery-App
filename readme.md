## QuickKart Local Grocery Delivery App 

## Grocery List Feature

This feature introduces a clean and organized **Grocery List** where items are grouped by category and can be filtered using a **search bar**.

### Features Implemented

- **GroceryList Component**
  - Displays grocery items grouped by category.
  - Implemented using **SectionList** for efficient rendering.
- **Grouped Data**
  - Added mock data for:
    - Fruits
    - Vegetables
    - Dairy
    - Snacks
  - Each section shows a header (e.g., **"Fruits"**) with items listed below.
- **Search Functionality**
  - Search bar added to filter items by name.
  - Clearing the search resets the full list.
- **UI/UX**
  - Clean, simple, scrollable design.
  - Smooth rendering and organized layout.

## How to Run the App

### 1. Install Expo CLI
```bash
npm install -g expo-cli
```

### 2. Install Dependencies
```bash
npm install
```

### 3. Run the App
```bash
npm start
```
